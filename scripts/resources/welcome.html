Welcome to Figma Discord Rich Presence

This installer will set up the Figma Discord RPC client on your Mac.

It will install the tool to /usr/local/bin and configure it to run automatically on login.

Important: macOS requires Accessibility permission so the app can read Figma window titles.

After installation, go to System Settings → Privacy & Security → Accessibility and add:

/usr/local/bin/figma-rpc
