Quick setup

  1. 1

    Download installer

    Pick your operating system and run the downloaded file.

  2. 2

    Allow OS prompt

    If your system warns about an unverified publisher, continue manually.

  3. 3

    Open Figma and Discord

    Keep both apps running and your Rich Presence should update in seconds.

Note: Windows SmartScreen or macOS Gatekeeper may show an unverified publisher warning. This is expected for this unsigned installer.
